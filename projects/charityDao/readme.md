# Charity DAO

慈善 DAO

## 项目概述

一个为编程小白/普通人准备的，为了慈善目的而发行 token / NFT 的平台。

## 技术架构

前端：

-   Vite
-   React + React Router
-   Wagmi
-   Typescript

后端：

-   Foundry
-   Solidity
